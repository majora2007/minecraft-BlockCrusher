What is BlockCrusher?

BlockCrusher is a Bukkit plugin which operates as an API for the popular Minecraft game. BlockCrusher extends the functionality of Pistons in Minecraft to allow blocks to break when pushed against a block which is un-pushable. Instead of not pushing any more blocks, the plugin allows for blocks to break naturally.

Configuration

The plugin allows for three different methods of breaking blocks:

  1. First - The block being pushed is broken.
  2. Last - The block being pushed against an un-pushable is broken.
  3. All - All blocks from the block being pushed to the last block are broken.
